    Contoso Software Solutions¶

    Welcome to the open source solutions of Contoso. Here you can find our Microsoft 365 integrations that we use to drive our business and enable our customers to do great things.

    Contoso Releases Secure SharePoint Document Generation Solution¶

    Our new SharePoint integration adds a button to document libraries enabling secure remote document generation on behalf of the current user. Read more about how we did it.

    Contoso Announces Release of Markdown File Handler for Microsoft 365¶

    The new file handler solution allows our customers to easily create, edit, and preview Markdown files using our existing SaaS services. We are excited to introduce this integration with OneDrive and SharePoint allowing us to seamlessly embed our custom functionality alongside the standard UI. Read more about how we did it.
